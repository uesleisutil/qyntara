# ROICalculator Component

## Overview

The `ROICalculator` component calculates and displays return on investment (ROI) metrics for the ML system, helping stakeholders justify costs and understand the value generated by the system.

## Features

### Core Functionality (Requirements 20.1-20.8)

1. **Portfolio Value Input** (20.2)
   - Configurable portfolio value managed by the system
   - Editable input with save/cancel functionality
   - Default value provided from data

2. **Value Generation Calculation** (20.3)
   - Calculates value generated as: `alpha * portfolio_value`
   - Uses actual alpha performance from the ML system
   - Displays total value generated over time

3. **ROI Calculation** (20.1, 20.4)
   - Formula: `ROI = ((value_generated - costs) / costs) * 100`
   - Displays ROI as percentage
   - Shows both daily and overall ROI metrics

4. **ROI Trend Visualization** (20.5, 20.6)
   - Line chart showing ROI over time
   - Dual-axis chart with ROI percentage and monetary values
   - Area charts for value generated and costs
   - Reference lines for target threshold and break-even

5. **Target Comparison** (20.7)
   - Compares current ROI against target threshold (default 200%)
   - Visual status indicators (excellent/good/fair/poor)
   - Progress bar showing ROI vs target
   - Status-based alerts and recommendations

6. **Break-Even Analysis** (20.8)
   - Calculates minimum portfolio size needed for break-even (ROI = 0%)
   - Calculates portfolio size needed to reach target ROI
   - Shows current portfolio position relative to thresholds
   - Provides interpretation guidance

## Props

```typescript
interface ROICalculatorProps {
  data: ROIData | null;
  isLoading?: boolean;
}

interface ROIData {
  roi?: {
    daily_metrics?: ROIDataPoint[];
    target_threshold?: number;
    default_portfolio_value?: number;
  };
}

interface ROIDataPoint {
  date: string;
  alpha: number;              // Model alpha performance (%)
  totalCost: number;          // Total daily cost
  portfolioValue?: number;    // Optional pre-calculated portfolio value
  valueGenerated?: number;    // Optional pre-calculated value
  roi?: number;              // Optional pre-calculated ROI
}
```

## Calculations

### Value Generated
```
value_generated = (alpha / 100) * portfolio_value
```

### ROI
```
roi = ((value_generated - total_cost) / total_cost) * 100
```

### Break-Even Portfolio
```
break_even_portfolio = total_cost / (alpha / 100)
```
This is the minimum portfolio value where ROI = 0%

### Target Portfolio
```
target_portfolio = ((target_roi / 100) * cost + cost) / (alpha / 100)
```
This is the portfolio value needed to achieve the target ROI

## Status Levels

The component categorizes ROI into four status levels:

- **Excellent** (🎯): ROI ≥ target threshold
- **Good** (✓): ROI ≥ 75% of target
- **Fair** (⚠️): ROI ≥ 50% of target
- **Poor** (❌): ROI < 50% of target
